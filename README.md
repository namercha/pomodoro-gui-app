# pomodoro-gui-app
Python app with a 25 min timer and 5 min breaks that can reset with user input.

Creates a 25 min timer, then a 5 min break.
After four 25 min sessions, a 20 min break is added.

Example:
25 min work
5 min break
25 min work
5 min break
25 min work
5 min break
25 min work
20 min break
